# Tabzer Game

Projeto inicial para Unity 6, estruturado como um jogo 2D de acao/plataforma com assets gerados.

## Como abrir

1. Abra o Unity Hub.
2. Clique em **Add** e selecione esta pasta: `c:\Users\Matheus\Documents\tabzerGame`.
3. Abra com qualquer editor Unity 6 / 6000.x instalado.
4. No editor, use **Tabzer > Create Starter Scene** para gerar `Assets/Scenes/Main.unity`.

## Assets organizados

- `Assets/Art/UI`: mockup de HUD e atlas visual de interface.
- `Assets/Art/Environment`: cenario de sala/escritorio e tiles/partes do ambiente.
- `Assets/Art/VFX`: efeitos de ataque, impacto, magia e particulas.
- `Assets/Art/Props`: props de escritorio, canecas, computador, teclado e itens.
- `Assets/Art/Enemies`: spritesheets do inimigo escuro.
- `Assets/Art/Player`: spritesheets do personagem principal.
- `Assets/Art/DesignReferences`: telas de referencia do modo Focus Quest.
- `Docs/asset_contact_sheet.png`: folha de contato com todos os PNGs originais.

## Proximo passo recomendado

Depois que o Unity importar os arquivos, abra o Sprite Editor nos spritesheets de player/inimigo e fatie as animacoes. Os scripts iniciais ja deixam um personagem 2D com movimento, pulo e uma camera simples.
